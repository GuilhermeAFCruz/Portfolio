JavaScript Calculator

Calculadora simples

Esse projeto é uma simples calculadora utilizando HTML, CSS e JavaScript Vanilla.

A ideia deste projeto é utilizar essas tecnologias para montar um simples aplicativo web de calculadora com uma interface funcional e manipulação de elementos.

Este projeto foi realizado através da aula gratuita disponível no canal Kevin Powell.
